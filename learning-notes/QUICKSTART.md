# 🚀 快速开始 - 5分钟运行你的第一个聊天应用

## 步骤 1️⃣: 获取 Sendbird Application ID

1. 打开浏览器,访问: https://dashboard.sendbird.com/
2. 点击 **Sign up** 注册账号(或登录已有账号)
3. 点击右下角的 **Create +** 按钮
4. 填写应用信息:
   - **Application name**: 输入 `我的聊天应用`
   - **Product Type**: 选择 `Chat`
   - **Region**: 选择 `Singapore` (亚洲用户) 或 `Oregon` (美国用户)
5. 点击 **Confirm** 创建应用
6. 在应用列表中点击刚创建的应用
7. 复制 **Application ID** (类似 `ABCD1234-EFGH-5678-IJKL-9012MNOPQRST`)

---

## 步骤 2️⃣: 配置项目

1. 在 Android Studio 中打开项目
2. 在左侧项目面板中找到: `app/src/main/java/com/example/myapplication/ChatApplication.kt`
3. 双击打开文件
4. 找到第 21 行的 `APP_ID`:

```kotlin
private const val APP_ID = "YOUR_SENDBIRD_APP_ID"
```

5. 替换为你复制的 Application ID:

```kotlin
private const val APP_ID = "ABCD1234-EFGH-5678-IJKL-9012MNOPQRST"
```

6. 保存文件 (Ctrl+S 或 Cmd+S)

---

## 步骤 3️⃣: 运行应用

### 选项 A: 使用 Android 模拟器

1. 点击顶部工具栏的 **Device Manager** 图标
2. 如果没有模拟器,点击 **Create Device**
   - 选择 **Phone** > **Pixel 5**
   - 点击 **Next**
   - 选择系统镜像(推荐 **S** API 31)
   - 点击 **Next** > **Finish**
3. 在设备列表中点击 ▶️ 启动模拟器
4. 等待模拟器完全启动
5. 点击 Android Studio 顶部的绿色 ▶️ **Run** 按钮

### 选项 B: 使用真实 Android 设备

1. 在手机上启用开发者选项:
   - 进入 **设置** > **关于手机**
   - 连续点击 **版本号** 7次
   - 返回设置,找到 **开发者选项**
   - 开启 **USB 调试**
2. 用 USB 线连接手机和电脑
3. 手机上会弹出提示,点击 **允许 USB 调试**
4. 在 Android Studio 顶部设备选择框中选择你的手机
5. 点击绿色 ▶️ **Run** 按钮

---

## 步骤 4️⃣: 测试聊天功能

### 单人测试

1. 应用启动后,你会看到登录界面
2. 输入用户 ID: `tom`
3. 输入频道名称: `测试聊天室`
4. 点击 **进入聊天室**
5. 等待几秒连接成功
6. 在底部输入框输入消息: `Hello Sendbird!`
7. 点击发送按钮 ✉️
8. 你会看到消息出现在聊天界面右侧(蓝色气泡)

### 多人测试 🎉

**方法 1: 使用两个模拟器**

1. 在 Device Manager 中创建第二个模拟器
2. 同时运行两个模拟器
3. 在第一个设备上:
   - 用户 ID: `tom`
   - 频道: `公共聊天室`
4. 在第二个设备上:
   - 用户 ID: `jerry`
   - 频道: `公共聊天室` (必须相同!)
5. 在任一设备发送消息,另一个设备会实时收到!

**方法 2: 模拟器 + 真机**

1. 在模拟器上用 `tom` 登录
2. 在真机上用 `jerry` 登录
3. 确保两者进入相同的频道名称
4. 开始聊天!

## 🐛 遇到问题?

### 应用无法安装

**原因**: Gradle 构建失败

**解决**:
1. 点击菜单 **Build** > **Clean Project**
2. 点击菜单 **Build** > **Rebuild Project**
3. 等待构建完成后重新运行

### 应用闪退

**原因**: 未配置 APP_ID

**解决**:
1. 检查 `ChatApplication.kt` 中是否替换了 `APP_ID`
2. 确保 Application ID 格式正确
3. 查看 Logcat(Android Studio 底部)的错误信息

### 连接失败

**错误信息**: "连接失败: Invalid app ID"

**解决**:
- 确认 Application ID 复制正确
- 确认没有多余的空格或引号
- 重新从 Sendbird Dashboard 复制

### 消息发送失败

**原因**: 网络问题或未进入频道

**解决**:
1. 确认设备/模拟器有网络连接
2. 查看 Logcat 的详细错误信息
3. 尝试退出重新进入频道

### 看不到日志

**解决**:
1. 点击 Android Studio 底部的 **Logcat** 标签
2. 在搜索框中输入: `ChatApplication`
3. 调整日志级别为: **Debug** 或 **Verbose**

## 📱 查看 Sendbird Dashboard

在 Dashboard 中可以:

1. **查看消息记录**:
   - 点击左侧 **Chat** > **Messages**
   - 可以看到所有发送的消息

2. **查看用户列表**:
   - 点击左侧 **Users**
   - 可以看到 `tom`, `jerry` 等用户

3. **查看频道列表**:
   - 点击左侧 **Chat** > **Open channels**
   - 可以看到创建的频道

4. **发送测试消息**:
   - 在 Dashboard 中也可以向频道发送消息
   - 应用会实时收到
