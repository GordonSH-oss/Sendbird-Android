# ✅ 每日学习检查清单

> 每天学习时使用这个清单,确保不遗漏任何重要步骤

---

## 📋 学习前 (5分钟)

- [ ] 打开今天的 `day-X.md` 文件
- [ ] 阅读今日学习目标
- [ ] 浏览任务清单
- [ ] 准备好 Android Studio
- [ ] 准备好笔和笔记本(可选)

---

## 📚 学习中 (1.5-2小时)

### 理论学习
- [ ] 阅读指定的文档章节
- [ ] 理解核心概念
- [ ] 记录不理解的地方
- [ ] 查看相关代码示例

### 实践练习
- [ ] 完成代码实验
- [ ] 运行应用验证
- [ ] 修改代码测试理解
- [ ] 查看 Logcat 日志

### 任务跟踪
- [ ] 完成一个任务就打勾 ✅
- [ ] 遇到问题立即记录
- [ ] 记录解决方案
- [ ] 保存重要的代码片段

---

## 📝 学习后 (15-20分钟)

### 成果记录
- [ ] 填写"学习成果记录"部分
  - [ ] 粘贴完成的代码
  - [ ] 记录实验结果
  - [ ] 截图保存(可选)

### 问题记录
- [ ] 记录遇到的所有问题
  - [ ] 问题描述
  - [ ] 错误信息
  - [ ] 解决方案
  - [ ] 学到的经验

### 思考题
- [ ] 回答所有思考题
- [ ] 用自己的话解释概念
- [ ] 不确定的标记出来,第二天查找答案

### 今日收获
- [ ] 列出掌握的技能
- [ ] 列出理解的概念
- [ ] 记录新学的术语
- [ ] 写下心得体会

### 自我评价
- [ ] 给理解程度打星 ⭐
- [ ] 给完成度打星 ⭐
- [ ] 记录需要加强的方面
- [ ] 写下今天的感受

---

## 📊 进度更新 (5分钟)

### PROGRESS.md 更新
- [ ] 在对应的 Day 后面标记 ✅
- [ ] 记录今天的学习时长
- [ ] 记录完成度(%)
- [ ] 更新"完成天数"计数

### 成就检查
- [ ] 查看是否解锁新成就
- [ ] 在 PROGRESS.md 中标记成就

---

## 🔄 第二天准备 (5分钟)

### 今天总结
- [ ] 快速浏览今天的笔记
- [ ] 确认所有任务都完成了
- [ ] 确认所有部分都填写了

### 明天预习
- [ ] 打开明天的 `day-X.md`
- [ ] 快速浏览学习目标
- [ ] 了解需要完成的任务
- [ ] 心理准备

---

- [ ] 规划下周重点

### 每周日额外检查

- [ ] 准备下周学习

## 🎯 记住

### 学习的目的
不是为了完成任务,而是为了:
- 真正理解概念
- 掌握实用技能
- 建立知识体系
- 成为 Android 开发者

### 不要担心
- ❌ 今天没完全理解 → 明天可以复习
- ❌ 代码出错了 → 正常的,学习过程
- ❌ 任务没全做完 → 第二天补上
- ❌ 进度比计划慢 → 理解比速度重要

### 最重要的是
- ✅ 每天都有进步
- ✅ 保持学习连续性
- ✅ 享受学习过程
- ✅ 相信自己能做到
