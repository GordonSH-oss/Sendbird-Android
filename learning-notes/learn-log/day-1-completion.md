## 🎓 今天学到的关键概念

### 1. Android 基础
- **Application 类**: 应用的入口点,用于全局初始化
- **Activity**: 代表一个界面/屏幕
- **AndroidManifest.xml**: 应用配置文件

### 2. 开发工具
- **Android Studio**: IDE 的基本使用
- **Logcat**: 查看应用日志
- **模拟器/真机**: 运行和调试应用

### 3. 项目结构
```
app/src/main/
├── java/              # Kotlin 源代码
├── res/               # 资源文件
└── AndroidManifest.xml # 配置文件
```

### 4. Sendbird 集成
- 获取 Application ID
- 初始化 SDK
- 连接用户
- 创建频道
- 发送消息
